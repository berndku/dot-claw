MakerSpace: DotClaw – Rebuild OpenClaw with C# and Microsoft Agent Framework

From chat to agent: Build a personal AI assistant that actually does things 🤖🦞

OpenClaw is an open-source personal AI assistant — it lives on your machine, has its own personality, remembers who you are, and can read files, run commands, and write code. In this MakerSpace session, we'll rebuild it from scratch in C# using the Microsoft Agent Framework and GitHub Copilot.

🎤 Speaker:
The session will be facilitated by Bernd Kullick, who will guide you through building a personal AI agent from scratch — no C# expertise required, GitHub Copilot writes the code for you.

🧠 What to Expect:
Understand how agent loops work, build autonomous tool calling with the Microsoft Agent Framework, give your agent a name and personality, and chat with it on your phone via Telegram — all in 60 minutes.
